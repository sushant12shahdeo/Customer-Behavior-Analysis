# Customer-Behavior-Analysis
Analyzed customer purchasing behavior using SQL, Python, Excel, and Power BI. Cleaned and transformed data, performed exploratory analysis, built interactive dashboards, tracked key KPIs, and generated actionable insights on sales trends, customer segments, and product performance.

## Tools Used
- Python (Pandas, NumPy)
- SQL (MySQL)
- Power BI
- Excel

## Project Workflow
- Loaded and explored the dataset.
- Cleaned missing and duplicate data.
- Performed Exploratory Data Analysis (EDA).
- Executed SQL queries for business insights.
- Built an interactive Power BI dashboard.
- Created a business report and presentation.

## Key Insights
- Identified customer purchasing patterns.
- Analyzed sales and product performance.
- Tracked important KPIs.
- Generated actionable business recommendations.

## How to Run
1. Clone the repository.
2. Install the required Python libraries.
3. Run the Python scripts.
4. Execute the SQL queries.
5. Open the Power BI (.pbix) file.
